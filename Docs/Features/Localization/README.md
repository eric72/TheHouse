# Localisation

**Runtime :** `UTheHouseLocalizationSubsystem` (culture active, application aux textes) et `UTheHouseLanguageSaveGame` (persistance du choix).

**Pipeline éditeur / CLI :** cibles **`Game`** (`Config/Localization/Game_*.ini`), script maître **`Scripts/RunLocalizationStep.ps1`** (`-Step Gather|Compile|Export|Import|…`), raccourcis `GatherLocalization.ps1` / `CompileLocalization.ps1`.
